Work

Research Project - Data Acquisition, Fusion, and Visualization: During my academic journey, I led a research project that stands as a testament to my strong research and development skills. Developed a comprehensive air quality analysis system using machine learning techniques and exploratory data analysis. Collected, processed, and analyzed EPA-provided datasets from 1987 to 2022, employing SVM, Random Forest, and Decision Tree algorithms. Conducted data pre-processing, transformation, and model evaluation, emphasizing MSE and RMSE. Collaborated with Professor Stefan robila, contributing to impactful research.

E-commerce System implemented on University Servers (DATABASE) Led the development of an advanced website, seamlessly integrating e-commerce features and a streamlined human resources management system for efficient employee tracking. Empowered users to register customer and employee accounts, optimizing business operations

Associate system engineer - ACCENTIQA: As an entry-level Associate System Engineer, I diligently contributed to system development, provided adept technical support, maintained meticulous system documentation, and collaborated effectively within cross-functional teams. Committed to upholding industry standards, I prioritized excellence in project delivery and consistently delivered exceptional customer service.

Sign to Speech Converter Using Web Camera: Another significant project I undertook was the development of a 'Sign to Speech Converter' tool. This tool is designed to assist differently-abled individuals by using a web camera and speech modules to translate sign language into speech. The project involved working with large datasets and implementing machine learning algorithms to achieve an impressive accuracy rate of 86%.

Auto Dimming in Automobiles: In a project focused on automotive safety, I designed and implemented a system that addressed the issue of headlight glare. We placed sensors beside vehicle headlights to check and reduce headlight intensity, thereby preventing glare and enhancing road safety. The project utilized an LDR sensor and Arduino UNO microprocessor, achieving an impressive accuracy rate of 93%.
